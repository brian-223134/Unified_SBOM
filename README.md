# 사용 방법
- docker을 실행한 이후에 아래의 명령어를 powershell에 입력하면 됩니다.
- Windows의 경우 WSL2와 docker의 연결 설정이 완료되어야 실행이 가능합니다.
  - Linux OS는 Ubuntu를 추천합니다. (LTS 버전, 21 이상 추천)
``` powershell
docker-compose up --build
```
